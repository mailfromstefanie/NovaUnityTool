OPNIEUW OPBOUWEN – ALS ALLES KWIJT IS
=====================================

Scenario A – Je hebt deze GitHub nog
1. Kopieer Editor/StefTools/ naar je Unity-project:
   Assets/Editor/StefTools/
2. Start Unity, open het menu: STEFSTOOLS -> Nova Export (simpel)
3. Klik “Nieuw project starten”.
4. Upload de gemaakte zip + Zet-in-chat.txt in een nieuwe AI-chat.
5. Vraag: “Vul mijn Projectplan.txt aan en geef de eerste drie stappen.”
6. Vervang jouw Projectplan.txt met de nieuwe versie uit de chat.

Scenario B – Zelfs deze repo ben je kwijt
- Vraag Nova: “Maak de Unity-exporttool opnieuw” en geef je wensen door.
- Daarna deze repo opnieuw maken en de stappen hierboven volgen.
